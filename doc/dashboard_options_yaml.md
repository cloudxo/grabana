# Dashboard options

Generic options describing the dashboard's behavior.

```yaml
title: Awesome dashboard
editable: true
shared_crosshair: true
tags: [generated, yaml]
auto_refresh: 10s

time: ["now-6h", "now"]
timezone: utc # valid values are: utc, browser, default
```
